# Godot Game Setup

## Requirements
- Godot Engine 4.5 or later

## Setup Instructions
1. Open **Godot 4.5**.
2. Click **Import**.
3. Navigate to `/path-to/game-of-power` and select `project.godot`.
4. Click **Import & Edit** to open the project.

## Running the Game
1. Open the project in Godot 4.5
2. Press F5 or click the Play button

## Prebuilt Downloads
- Google Drive: https://drive.google.com/drive/folders/1NvQ-NhF5FCd0jz5km1tN6qBmyi1YVsrX?usp=sharing

### Android (APK)
- Download the APK from Google Drive.
- Enable **Install unknown apps** on your Android device.
- Install and launch the game.

### Windows (EXE)
- Download the Windows build (`.exe` and `.pck` files) from Google Drive.
- Make sure both files are in the **same folder**.
- Double-click the `.exe` file to launch the game.

## Notes
- The Windows build requires the `.exe` and `.pck` files to remain together.
- The game runs offline on both platforms.
